# Hanu-Tech – Facade Fabrication Company Website

![Hanu-Tech](./assets/logo.png)

## 🌐 Overview

Hanu-Tech is a modern, responsive business website developed for a facade fabrication company specializing in architectural exterior solutions. The website showcases completed projects, services, trusted clients, company information, and contact details with a clean and professional user interface.

## ✨ Features

- Fully Responsive Design
- Modern UI/UX
- Hero Section
- About Company
- Services Section
- Project Gallery
- Trusted Clients
- Contact Form
- Google Maps Integration
- Smooth Animations
- SEO Optimized
- Fast Loading Performance

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Bootstrap 5
- EmailJS
- Google Maps
- Font Awesome
